Navigation Web App
Aim
Google maps only utilize roads and walkway in public areas, how about places that need more specifics? e.g. Zoo, park, etc. This navigation app aims to replace handheld maps in these areas, moving from point A to point B will be more smooth and efficient. 
Features
This Web App uses Users' GPS and checks accuracy before providing the map with directions to the user. After a destination is entered, the shortest route will appear
Technical Details
This application was developed using HTML, CSS, JavaScript
Involvement
I mainly developed the front-end and the map animation with the shortest route
